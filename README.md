# Walmart-supermarket-sales-Insights--python-visualization
Python-based Exploratory Data Analysis (EDA) project on Walmart sales data, involving data cleaning, preprocessing, and in-depth analysis to uncover sales trends, customer purchasing behavior, and key business insights. The project utilizes Pandas, NumPy, Matplotlib, and Seaborn for data manipulation, visualization, and statistical analysis.

This project focuses on performing a comprehensive Exploratory Data Analysis (EDA) on the Walmart Sales dataset using Python. The main objective is to understand the structure of the data, clean and preprocess it, and extract meaningful insights that can support business decision-making. The dataset contains transactional details such as product lines, customer demographics, payment methods, and sales performance, making it suitable for analyzing real-world retail patterns.

The analysis involves multiple steps including data cleaning, handling missing values, removing duplicates, and feature engineering by creating new time-based variables. Various analytical techniques such as univariate, bivariate, and multivariate analysis are applied to explore relationships between different variables. Visualization tools like Matplotlib and Seaborn are used extensively to represent the data through bar charts, line graphs, pie charts, histograms, box plots, and heatmaps for better understanding and interpretation.

Through this project, key insights are generated regarding customer purchasing behavior, popular product categories, preferred payment methods, and overall sales trends across different cities and time periods. These findings can help businesses improve their marketing strategies, optimize inventory management, and enhance customer experience. Overall, this project demonstrates strong data analysis, visualization, and problem-solving skills using Python.
